Параметры процедуры и возвращаемое значение

is_bigger_than :: proc(number: int, compare_to: int) -> bool {
    return number > compare_to
}
  • is_bigger_than — процедура
  • number и compare_to — параметры (целые числа)
  • bool — возвращаемое значение (истина/ложь)

Struct

Группирует переменные в единую структуру. Это упрощает работу, когда данные объединены по смыслу, а не хранятся отдельными кусочками. 
Rectangle :: struct {
    x: f32,
    y: f32,
    width: f32,
    height: f32,
}
